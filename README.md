# Volkswagen Botswana Dealership

![VW Botswana](https://img.shields.io/badge/Volkswagen-Das%20Auto-blue)

A fast, modern dealership website for Volkswagen in Botswana featuring the iconic "Das Auto" motto.

## Features
- ⚡ Responsive design
- 🚗 Showroom inventory display
- 🇧🇼 Botswana-specific branding
- 🎯 "Das Auto" motto integration
- 🔧 Fast loading and optimized

## Technologies
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

1. Clone this repository
2. Open `index.html` in your browser
