## 🔍 Overview

A modern, fully interactive dealership website for **Volkswagen Botswana**, featuring the iconic "Das Auto" motto. This website provides an engaging user experience with functional links, interactive prompts, and a professional automotive brand presence.

The site is designed to showcase Volkswagen's commitment to quality, performance, and service across Botswana's major cities.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Responsive Design** | Fully mobile-friendly layout |
| **Interactive Prompts** | User-friendly dialogue boxes for actions |
| **Test Drive Booking** | Simple form to schedule test drives |
| **Inventory Display** | Showcase of available VW models |
| **Location Stamps** | Information for Gaborone, Francistown, Maun |
| **Social Media Links** | Connect with VW Botswana on social platforms |
| **Fast Loading** | Optimized for quick page loads |
| **Brand Consistency** | Volkswagen signature colors and styling |

---

## 🎯 Interactive Elements

| Element | Action | Result |
|---------|--------|--------|
| **Test Drive** | Click header button | Prompts for name & phone, confirms scheduling |
| **Inventory** | Click 🔍 inventory button | Asks for model preference, confirms availability |
| **Contact** | Click 📞 contact button | Prompts for name & message, confirms delivery |
| **Car Models** | Click any inventory item | Shows car details, offers test drive |
| **Feature Cards** | Click VW performance/Local service/etc. | Displays feature information |
| **Location Stamps** | Click city names (Gaborone/etc.) | Shows dealership hours and contact |
| **Botswana Flag** | Click 🇧🇼 | Displays VW Botswana history |
| **"Run Fast"** | Click ⚡ run fast | Shows brand motto and values |
| **Social Media** | Click Instagram/Facebook/WhatsApp/X | Opens social pages in new tab |
